# AML Monitoring Lab

> 법학 전공자 관점에서 자금세탁방지(AML) 업무의 "법규 → 데이터 → AI 보조" 흐름을 직접 손으로 구현해본 학습 목적 프로젝트입니다.

## 구성 (작업 진행 중)

본 레포는 두 개의 독립 모듈로 구성할 예정입니다.

- **Module 1: Alert Triage Demo** — 가상 거래 데이터를 기반으로 AML 의심거래 탐지 룰을 SQL로 구현하고 위험 점수를 산출.
- **Module 2: Regulatory Monitor** — FATF/OFAC/금융위/FIU의 규제·제재 공지를 수집해 한국어로 요약하는 소형 에이전트.

## 설계 원칙 (예정)

- Rule-based + Explainable
- Human-in-the-loop
- Synthetic data only
- Source-traceable
